Perspectiva de valores

Para pensar em algum tipo de progressão, tomaremos como base a média do R$/hora de trabalho:


  • 2013 - ~R$6/hora => ~120h (6h/d) => ~R$720,00
  • 2014 - ~R$13/hora => ~120h (6h/d) => ~R$1800,00

  • 2015 - ~R$20/hora => ~120h (6h/d) => ~R$2400,00

  • 2016 - ~R$30/hora => ~120h (6h/d) => ~R$3600,00

  • 2017 - ~R$35/hora => ~120h (6h/d) => ~R$4200,00

Esses R$/hora foram baseados na nossa experiencia pessoal dentro do contexto de Bauru. Em São Paulo, essa já seria uma média bem baixa.
